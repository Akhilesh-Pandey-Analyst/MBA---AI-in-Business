# 5.4 IQR-based capping (Winsorization) - clean, single source of truth
import numpy as np
import pandas as pd

df_capped = df.copy()  # df is your original loaded DataFrame

changed_counts = {}

for col in df_capped.columns:
    # Skip categorical columns if any were set as dtype 'category'
    if str(df_capped[col].dtype) == 'category':
        changed_counts[col] = 0
        continue

    Q1 = df_capped[col].quantile(0.25)
    Q3 = df_capped[col].quantile(0.75)
    IQR = Q3 - Q1
    lower = Q1 - 1.5 * IQR
    upper = Q3 + 1.5 * IQR

    before = df_capped[col].copy()
    df_capped[col] = np.where(df_capped[col] < lower, lower, df_capped[col])
    df_capped[col] = np.where(df_capped[col] > upper, upper, df_capped[col])
    changed_counts[col] = int((before != df_capped[col]).sum())

# Display a neat summary table of how many values were capped per column

def cap_outliers_iqr(s):
    q1 = s.quantile(0.25)
    q3 = s.quantile(0.75)
    iqr = q3 - q1
    lower = q1 - 1.5 * iqr
    upper = q3 + 1.5 * iqr
    return s.clip(lower, upper)

num_cols = [c for c in df.columns if c != "CHAS"] 
df[num_cols] = df[num_cols].apply(cap_outliers_iqr)

def iqr_bounds(s):
    q1, q3 = s.quantile(0.25), s.quantile(0.75)
    iqr = q3 - q1
    lower, upper = q1 - 1.5*iqr, q3 + 1.5*iqr
    return lower, upper

changed_counts = {}
for c in [col for col in df.columns if col != "CHAS"]:
    low, up = iqr_bounds(df[c])
    changed_counts[c] = int((df[c] == low).sum() + (df[c] == up).sum())

Phase 2: Business Narrative Summary¶

The exploratory analysis of the Boston Housing dataset highlights four major factors that strongly influence residential property values:

  1. Structural Features (RM): Homes with more rooms consistently show higher prices. This is a key selling point for developers and sellers, indicating that structural improvements and larger layouts add measurable economic value.

  2. Neighborhood Safety (CRIM): Higher crime rates correspond to lower property prices. For urban planners and policymakers, this insight reinforces the long-term benefit of safety initiatives in improving housing market performance.

  3. Education Quality (PTRATIO): Lower pupil–teacher ratios are associated with higher home values. This KPI is important for families and directly affects how real estate companies position properties located near well-rated schools.

  4. Socioeconomic Indicators (LSTAT): Neighborhoods with lower percentages of lowerincome residents tend to have higher property values. This reinforces the role of socioeconomic development in shaping housing demand.

Implications for Stakeholders¶

  • Real Estate Developers: Highlight room count, school quality, and safety metrics in marketing campaigns.

  • Policy Makers: Investments in education, safety, and community upliftment can improve long-term property values.

  • Buyers & Investors: These indicators provide a research-backed basis for understanding neighborhood value differences.

The cleaned dataset and EDA outputs now form a strong foundation for Phase 3, where these insights will be consolidated into a comprehensive business report with ethical onsiderations and strategic recommendations.
